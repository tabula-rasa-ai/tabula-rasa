Query Parameters

ParameterTypeDefaultDescription
promptstring—Math expression ending with = (e.g. "12+34=")
temperaturefloat0.0Generation temperature (0=greedy, >0 adds randomness)
top_kint5Top-K sampling threshold
checkpointstringlatestPath to a specific checkpoint file
targetfloat50Auto-train target accuracy %
budgetint50000Auto-train max steps budget
opstring—Operation: add|sub|mul|div
quickboolfalseQuick mode (500 steps, 2K samples)
ewcboolfalseEnable Online EWC during training
ampboolfalseEnable mixed precision (requires CUDA)
grad_accumint1Gradient accumulation steps

Example Workflows

1. Train and query a specialist

# Train addition (quick mode)
curl -X POST http://localhost:8000/api/train/start \
  -H "Content-Type: application/json" \
  -d '{"op": "add", "quick": true}'

# Check status
curl http://localhost:8000/api/train/status

# Query the trained model
curl -X POST http://localhost:8000/generate \
  -H "Content-Type: application/json" \
  -d '{"prompt": "12+34="}'
# {"result": "46", ...}

2. Train with EWC (continual learning)

# Train addition first
curl -X POST http://localhost:8000/api/train/start \
  -H "Content-Type: application/json" \
  -d '{"op": "add"}'

# Train subtraction while preserving addition
curl -X POST http://localhost:8000/api/train/start \
  -H "Content-Type: application/json" \
  -d '{"op": "sub", "ewc": true}'

3. Auto-train loop

# Let the system train all operations automatically
curl -X POST http://localhost:8000/api/train/auto \
  -H "Content-Type: application/json" \
  -d '{"target": 70, "budget": 10000}'

# Monitor progress
curl http://localhost:8000/api/train/auto/status

4. Export and import specialists

# Export addition specialist
curl -X POST http://localhost:8000/api/export \
  -H "Content-Type: application/json" \
  -d '{"op": "add"}'
# exports/add.zip created

# Import from a shared URL
curl -X POST http://localhost:8000/api/import \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com/specialist.zip"}'

API server: api_server.py (port 8000) — serve.py (port 8000, older version).
All endpoints accept Content-Type: application/json. CORS headers set for all origins.
