Common Transformer Issues

  • NaN/Inf loss — Gradient explosion, bad init, or lr too high. Reduce lr, add gradient clipping, check for log(0)
  • Loss plateau > 20% — Wrong max_digits, reversed digits off, or Causal-Carry Mismatch. Check data format and model capacity.
  • Slow convergence — LR too low, batch size too small, or insufficient warmup steps.
  • Oscillating loss — LR too high (loss bounces) or batch noise (batch size too small for task complexity).
  • Sudden drop — Model found a good feature. Normal and expected. If followed by flat plateau, capacity may be saturated.
