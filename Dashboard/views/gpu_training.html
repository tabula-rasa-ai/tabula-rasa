📋 Quick Start

1
Select files & build
Check the files above and click Build Package. It creates tabula-rasa-gpu.tar.gz with all training scripts and configs.
2
Upload to GPU cloud
Rent a GPU pod (RunPod, Vast.ai), open the web terminal, and upload the zip:
# Upload via RunPod file manager or SCP
$ tar -xzf tabula-rasa-gpu.tar.gz
$ cd tabula-rasa-gpu
3
Run training
One command installs deps, prepares data, and trains all specialists on GPU:
$ bash setup_gpu.sh
# Or step by step:
$ pip install -r requirements.txt
$ python3 train_gpu.py
4
Download trained zips
Portable specialist zips land in /output/. Download via file manager and copy to your local specialists/math/<op>/

🧪 Quick CPU Test (validate before GPU)

Run a small training on your local CPU to verify the specialist works before spending money on GPU cloud.

Generalist: use Training Monitor page or terminal: python3 train.py

🤖 Select Specialists to Train

Choose which models to train on the GPU. The command will be generated automatically.

Selected: 5 models · 60,000 total steps · ~9 min on RTX 4080
# Generated command (run on GPU server):
$ python3 train_gpu.py --specialists generalist addition subtraction multiplication division --steps 60000 --batch 512 --reversed --loss-mask
